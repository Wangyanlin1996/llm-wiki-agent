## Summary

3GPP TS 28.622 defines the Generic Network Resource Model (NRM) Integration Reference Point (IRP) Information Service (IS) — the **foundational information model** for all 3GPP management specifications. It provides the core IOCs (Top, TopX, SubNetwork, ManagedElement, ManagedFunction, etc.) and common data types (GeoArea, TimeWindow, SchedulingTime, etc.) that domain-specific specs import and extend.

## Core IOC Hierarchy

```
Top_ (UIM, TS 28.620)
  └─ TopX (objectClass, objectInstance — mandatory for all IOCs)
      └─ Top (inherits Top_ + TopX — root for all conformant IOCs)
          ├─ SubNetwork (inherits Domain_ + TopX)
          ├─ ManagedElement (inherits ManagedElement_ + TopX)
          │   └─ ManagedFunction (abstract, for sub-classing)
          ├─ ManagementNode
          ├─ MeContext
          ├─ MnsAgent
          ├─ MnsRegistry
          │   └─ MnsInfo
          ├─ NtfSubscriptionControl
          ├─ ThresholdMonitor
          ├─ PerfMetricJob
          ├─ HeartbeatControl
          ├─ TraceJob
          ├─ VsDataContainer (vendor-specific data)
          ├─ ConditionMonitor
          ├─ Scheduler
          └─ SupportedNotifications
```
