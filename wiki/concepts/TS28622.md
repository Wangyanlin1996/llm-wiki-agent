## 概要

3GPP TS 28.622 定义了通用 Network Resource Model（NRM）Integration Reference Point（IRP）Information Service（IS）——所有 3GPP 管理规范的**基础信息模型**。它提供了核心 IOC（Top, TopX, SubNetwork→子网络, ManagedElement→管理单元, ManagedFunction 等）和通用数据类型（GeoArea, TimeWindow, SchedulingTime 等），领域特定规范在此基础上导入和扩展。

## 核心 IOC 层次

```
Top_ (UIM, TS 28.620)
  └─ TopX (objectClass, objectInstance — mandatory for all IOCs)
      └─ Top (inherits Top_ + TopX — root for all conformant IOCs)
          ├─ SubNetwork (inherits Domain_ + TopX)
          ├─ ManagedElement (inherits ManagedElement_ + TopX)
          │   └─ ManagedFunction (abstract, for sub-classing)
          ├─ ManagementNode
          ├─ MeContext
          ├─ MnsAgent
          ├─ MnsRegistry
          │   └─ MnsInfo
          ├─ NtfSubscriptionControl
          ├─ ThresholdMonitor
          ├─ PerfMetricJob
          ├─ HeartbeatControl
          ├─ TraceJob
          ├─ VsDataContainer (vendor-specific data)
          ├─ ConditionMonitor
          ├─ Scheduler
          └─ SupportedNotifications
```
