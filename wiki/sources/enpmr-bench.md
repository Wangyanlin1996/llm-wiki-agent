## 概要
ENPMR-Bench 为情感支持 agents 建立了 proactive memory retrieval（主动式记忆检索）的评测基准，评估 agents 应在何时、以何种方式主动回忆过往交互以提供情感支持性回应。

## 核心论点
- Proactive memory retrieval（主动式记忆检索）有别于被动检索
- 情感支持需要适时的无提示回忆
- 新 benchmark 填补了评估主动式 memory 使用方面的空白

## 关键引述
> "Proactive memory retrieval is distinct from passive retrieval" — 区别论点：主动式记忆检索与被动检索不同
