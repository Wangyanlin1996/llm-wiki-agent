## 概要

3GPP TS 28.622 定义了 Generic Network Resource Model (NRM) Integration Reference Point (IRP) Information Service——所有 3GPP 管理规范的基础信息模型。它定义了核心 IOCs（Top、TopX、SubNetwork、ManagedElement、ManagedFunction、ManagementNode、MeContext、MnsAgent 等）和通用数据类型（GeoArea、TimeWindow、SchedulingTime、PlmnId、IpAddr 等），由域特定 NRM 规范继承和复用（TS 28.541 用于 5G NRM、TS 28.556 用于 Policy、TS 28.312 用于 Intent 等）。

## 关键论点

- **Top IOC** 是所有符合 TS 32.102 的规范中 IOC 继承层级体系的根；它继承自 TopX（objectClass、objectInstance）和 Top_（来自 TS 28.620 UIM）
- **TopX IOC** 提供必选属性 objectClass 和 objectInstance——任何被管理对象的最小标识
- **SubNetwork IOC** 表示一组被管理实体；根 SubNetwork 实例包含所有其他实例；继承自 Domain_ 和 TopX
- **ManagedElement IOC** 表示电信设备或 TMN 实体；可包含 1..N 个 ManagedFunction 实例；继承自 ManagedElement_ 和 TopX
- **ManagedFunction IOC** 为抽象类（供子类化使用）；表示 ManagedElement 中包含的功能方面；继承自 Function_ 和 TopX
- **MnsAgent IOC** 表示 Service Based Management Architecture (SBMA) 中的 MnS 生产者；继承自 Top_ 和 TopX
- **VsDataContainer IOC** 允许将厂商特定数据附加到任何 IOC（被 Top 包含）
- **NtfSubscriptionControl IOC** 管理通知订阅，含 scope、filter 和接收地址；由 IntentReport 订阅机制引用
- **ThresholdMonitor IOC** 监控性能指标阈值，含滞后和方向性触发
- **HeartbeatControl IOC** 控制心跳通知发送以监控通信通道
- **TraceJob IOC** 表示 trace/MDT 控制和配置参数
- **PerfMetricJob IOC** 表示性能指标采集任务
- **MnsRegistry IOC + MnsInfo IOC** 表示 MnS 服务注册，用于能力发现
- 通用数据类型：GeoArea、TimeWindow、GeoCoordinate、SchedulingTime、TimeInterval、PlmnId、IpAddr、Host、PLMNInfo、AreaScope、Tai、NpnId、Scope、NodeFilter、VnfParameters、PeeParameters、TraceReference、ProcessMonitor
- 支持 TS 32.107/28.620 中的 FNIM（Federated Network Information Model）概念
- Release 20 版本——最新版，取代 Rel-14 IRP 框架版本用于 SBMA 部署

## 关键引述

> "All information object classes defined in all TS that claim to be conformant to TS 32.102 [2] and support the Federated Network Information Model (FNIM) concept shall inherit from Top." — Section 4.3.29.1，规定所有符合 TS 32.102 并支持 FNIM 的 IOC 必须继承自 Top

> "All information object classes defined in all TS that claim to be conformant to TS 32.102 [2] shall inherit from TopX." — Section 4.3.8.1，规定所有符合 TS 32.102 的 IOC 必须继承自 TopX

> "The present document specifies the Generic network resource information that can be communicated between an MnS producer and MnS consumer for telecommunication network management purposes, including management of converged networks and networks that include virtualized network functions." — Section 1，阐述本规范定义 MnS 生产者与消费者之间可传递的通用网络资源信息
