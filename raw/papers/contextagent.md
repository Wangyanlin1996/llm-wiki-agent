## Abstract
We introduce ContextAgent, the first context-aware proactive agent that incorporates extensive sensory contexts surrounding humans to enhance the proactivity of LLM agents. ContextAgent first extracts multi-dimensional contexts from massive sensory perceptions on wearables to understand user intentions. ContextAgent then leverages the sensory contexts and personas from historical data to predict the necessity for proactive services. When proactive assistance is needed, ContextAgent further automatically calls necessary tools to assist users unobtrusively. We curate ContextAgentBench, the first benchmark for evaluating context-aware proactive LLM agents, covering 1,000 samples across nine daily scenarios and twenty tools. ContextAgent outperforms baselines by achieving up to 8.5% and 6.0% higher accuracy in proactive predictions and tool calling.

## Key Contributions
- Introduces ContextAgent, the first context-aware proactive LLM agent with sensory perceptions
- Extracts multi-dimensional contexts from wearable sensory data to understand user intentions
- Predicts proactive service necessity using sensory contexts and historical personas
- Automatically calls tools for unobtrusive proactive assistance
- Curates ContextAgentBench (1,000 samples, 9 scenarios, 20 tools)
- Outperforms baselines by 8.5% in proactive predictions and 6.0% in tool calling
