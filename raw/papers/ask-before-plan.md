## Abstract
We introduce Proactive Agent Planning, requiring language agents to predict clarification needs based on user-agent conversation and agent-environment interaction, invoke external tools, and generate a plan. We establish the Ask-before-Plan benchmark dataset. We propose a multi-agent framework CEP (Clarification-Execution-Planning), with trajectory tuning and memory recollection mechanisms. Extensive evaluations validate the effectiveness.

## Key Contributions
- Introduces Proactive Agent Planning task for predicting clarification needs
- Establishes Ask-before-Plan benchmark dataset
- Proposes CEP multi-agent framework (Clarification-Execution-Planning)
- Incorporates trajectory tuning and memory recollection mechanisms
- Extensive evaluations validate effectiveness
