## Abstract
Current GUI agents operate under a reactive paradigm requiring explicit user instruction. PIRA-Bench introduces a Proactive Intent Recommendation Agent Benchmark for evaluating MLLMs on continuous, weakly-supervised visual inputs. Unlike reactive datasets, PIRA-Bench features complex trajectories with multiple interleaved intents and noisy segments with various user profile contexts, challenging agents to detect actionable events while fitting to user preferences. The paper also proposes PIRF, a memory-aware state-tracking framework that empowers MLLMs to manage multiple task threads and handle misleading visual inputs. PIRA-Bench serves as an initial step toward robust proactive GUI-based personal assistants.

## Key Contributions
- First proactive intent recommendation benchmark on GUI
- Complex interleaved intents + noisy segments
- User profile context integration
- PIRF memory-aware state-tracking framework
- Reactive→proactive paradigm transition
