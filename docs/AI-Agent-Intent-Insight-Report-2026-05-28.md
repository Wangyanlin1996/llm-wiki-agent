## 关键洞察

### 1. 跨方向共性：记忆→理解→推荐的三位一体

三个子方向均围绕 **长期记忆 + 用户画像建模 + 主动性** 收敛。智能体记忆提供存储基础设施（STM/MTM/LTM、混合记忆），意图理解提取隐式意图（IntPro 用户历史、IntentGrasp 基准、IST 定理），意图推荐实现主动行动（PIRF 状态追踪、DD-MM-PAS 闭环、Inner Thoughts 内隐层）。三者形成闭环：**记住 → 理解 → 主动推荐**。

### 2. 人因风险：主动不等于更好

BISE 2025（22引用）实验揭示：主动帮助反而降低用户能力自尊感。这与 3GPP Intent 的"声明式目标"理念存在张力——意图驱动强调自主性，但过度主动可能侵蚀用户自主性。设计需平衡 **Proactivity 与 User Autonomy**。

### 3. 评测多样化爆发

智能体记忆方向 2026 年涌现 6+ 个新评测基准（MemGym、ENPMR-Bench、MINTEval、EvoMemBench 等），意图理解有 IntentGrasp（12域49语料）。表明该领域从"方法创新"阶段进入"标准化评测"阶段。

### 4. LLM 意图理解能力严重不足

IntentGrasp 发现 17/20 LLM 在困难集上低于随机基线（15.2%），人类约 81%。定量验证了 Intent Signal Theory 的不可逆意图丢失定理——LLM 无法从 Prompt 恢复用户潜在意图（I\*）。

### 5. 与 3GPP 意图管理的关联

3GPP Intent Driven MnS 与 AI Agent 意图研究共享同一基础原则：**表达"要什么"而非"怎么做"**。Rule→Policy→Intent 的抽象层级与 Agent Reactive→Context-Aware→Proactive 的演进对偶。两者均将意图形式化为更高抽象层，使自主目标追求成为可能。
